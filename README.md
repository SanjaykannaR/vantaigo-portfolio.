Welcome to the official portfolio repository for **Vantaigo**, a forward-thinking 2025 startup dedicated to bridging the gap between human potential and technological innovation.

## 🚀 About Vantaigo
Founded in 2025, Vantaigo is a dual-stream tech firm specializing in:
1.  **HR Solutions:** Automating recruitment, payroll, and employee engagement using AI.
2.  **Software Solutions:** Building bespoke enterprise software, mobile apps, and cloud infrastructure for modern businesses.

## 🛠 Tech Stack
- **Frontend:** React.js / Next.js 15 (App Router), Tailwind CSS
- **Backend:** Node.js
- **Database:** MongoDB
